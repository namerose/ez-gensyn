## ðŸ“¥ Installation

1. **Go `cd $HOME` First**
```bash
cd $HOME
```
2. **Install `sudo`**
```bash
apt update && apt install -y sudo
```
3. **Install Git**
```bash
sudo apt install git
```
4. **do Clone Repo**  
```bash
git clone https://github.com/namerose/ez-gensyn.git
```
5. **run**
```bash
chmod +x ez-gensyn/install.sh && ./ez-gensyn/install.sh
```
